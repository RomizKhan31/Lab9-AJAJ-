2. INTRODUCTION
================================================================================

2.1 What is AJAX?
AJAX stands for Asynchronous JavaScript and XML. It is a set of web 
development techniques that allow web applications to fetch data asynchronously 
in the background without interfering with user interaction or page reloading.

Key characteristics of AJAX:
- Asynchronous Communication: Data is fetched without blocking user actions
- Partial Page Updates: Only specific parts of the page are updated
- No Page Reload: Users experience seamless interaction
- Improved Performance: Reduces bandwidth by loading only needed data
- Better User Experience: Eliminates traditional form submission delays

2.2 XMLHttpRequest (XHR)
XMLHttpRequest is the fundamental technology behind AJAX that enables:
- Making HTTP requests from JavaScript
- Handling server responses asynchronously
- Parsing JSON and XML data
- Real-time content updates

2.3 JSON Format
JSON (JavaScript Object Notation) is a lightweight data format that:
- Uses human-readable structure
- Supports nested objects and arrays
- Integrates seamlessly with JavaScript
- Reduces bandwidth compared to XML

2.4 Use Cases for AJAX
Common applications include:
- Live search suggestions
- Auto-save functionality
- Real-time notifications
- Dynamic dropdown menus
- Dashboard data updates
- Form validation without reload

8. CODE IMPLEMENTATION
================================================================================

8.1 JavaScript Function (AJAX)
```javascript
function loadDept(facultyCode) {
  if (facultyCode === "") {
    document.getElementById("output").innerHTML = "";
    return;
  }
  var xhr = new XMLHttpRequest();
  xhr.open("GET", "data.json", true);
  xhr.onreadystatechange = function () {
    if (xhr.readyState === 4 && xhr.status === 200) {
      var data = JSON.parse(xhr.responseText);
      
      var selectedFaculty = null;
      for (var i = 0; i < data.length; i++) {
        if (data[i].code === facultyCode) {
          selectedFaculty = data[i];
          break;
        }
      }
      
      if (selectedFaculty) {
        var html = "<h3>" + selectedFaculty.faculty + "</h3>";
        html += "<ul>";
        for (var j = 0; j < selectedFaculty.departments.length; j++) {
          html += "<li><strong>" + 
                  selectedFaculty.departments[j].name + 
                  "</strong> - Chairman: " + 
                  selectedFaculty.departments[j].chairman + 
                  "</li>";
        }
        html += "</ul>";
        document.getElementById("output").innerHTML = html;
      }
    }
  };
  xhr.send();
}
```

8.2 Event Handler
The select dropdown uses inline onchange event:
<select id="deptSelect" onchange="loadDept(this.value)">

This triggers the AJAX function whenever the user selects a faculty.
